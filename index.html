Password Generator

The Secure Password Generator is a lightweight yet powerful web-based tool designed to help users create strong, unpredictable, and secure passwords tailored to their personal or organizational needs. In a digital world where password breaches are increasingly common, this application empowers users with the ability to enhance their online security through the creation of robust passwords that adhere to industry best practices. This project emphasizes simplicity, usability, and security — allowing users to define password criteria and instantly generate passwords that are difficult to guess or crack, reducing the likelihood of successful brute-force or dictionary attacks.

Snake Game

The Classic Snake Game is a browser-based implementation of the timeless arcade game, built entirely using JavaScript, HTML, and CSS. This interactive project recreates the nostalgic gameplay of controlling a snake as it moves across the screen, grows by consuming food, and avoids collisions with itself or the walls. This project not only showcases fundamental programming skills but also demonstrates effective use of the DOM API, JavaScript-based animation, and responsive design principles, making it a complete and engaging browser experience.
